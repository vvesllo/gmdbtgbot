Запуск
```docker-compose up -d```

Просмотр логов
```docker-compose logs -f```

Остановка
```docker-compose down```

Пересборка
```docker-compose up -d --build```

Логи
```docker-compose logs -f bot```

Доступ в контейнер
```docker-compose exec bot bash```

Статус контейнера
```docker-compose ps```

Использование ресурсов
```docker stats```
